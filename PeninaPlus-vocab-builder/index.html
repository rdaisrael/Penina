About Penina Vocabulary Builder:
Penina Vocabulary Builder is a specialized, AI-assisted study tool designed to generate customized, highly accessible Hebrew and Aramaic vocabulary lists. It automatically fetches definitions and contextual sentences from classical sources or modern conversational Hebrew, applies selective vowelization, and offers granular control over typography and formatting to support diverse reading needs.

Credits:
• Developed by David Israel
• Utilizes the Gemini API
• Vowelization utilizes the Dicta API
• Hebrew biblical and Talmudic context sources selected from Sefaria
• Hebrew Dyslexia-friendly typography via Dyslexic-Kriah
• English Dyslexia-friendly typography via OpenDyslexic
